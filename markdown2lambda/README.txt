HOW TO USE:
Make a new folder, call it what you want
Put your markdown (.md) file in that folder. Rename the file 'template.md'
If you want to put images on your set, create a folder called 'media' and add in images
    in the markdown folder use relative links to the images, follow the template below exactly:
        ![pictureTag](media\IMAGEHERE)
        Only change the IMAGEHERE section. Do not change pictureTag
See the template for how to set out your questions
    Delimit questions with one hashtag 
    parts with two hashtags
Start your markdown file with an empty line (two spaces and a carrage return)
